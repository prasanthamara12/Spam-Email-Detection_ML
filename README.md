Email Spam Detection using Machine Learning is a project that classifies emails as spam or non-spam using Natural Language Processing (NLP) techniques. The system preprocesses email text, converts it into numerical features using TF-IDF Vectorization, and trains a Multinomial Naive Bayes model for accurate classification. Developed using Python, Pandas, Scikit-learn, and Google Colab, this project demonstrates the complete machine learning workflow, including data preprocessing, feature extraction, model training, evaluation, and real-time prediction of email messages. It provides an effective solution for automated email filtering and spam detection.
